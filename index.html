About Nova Health

Clinical decision screening prototype leveraging client-side machine learning and zero-network data isolation.

Nova Health

Nova Health is an interactive, browser-based clinical screening platform designed to assist users in understanding symptom correlations. By combining client-side machine learning execution with secure data isolation, we offer a fast, anonymous, and educational health monitoring workbench.

Project Objectives

To demonstrate the utility of lightweight machine learning classifiers executing in real-time within the web browser. Nova Health aims to promote health literacy and encourage early, structured discussions between patients and primary care physicians.

Data Confidentiality

We operate on a zero-trust model. Nova Health runs entirely within your browser's execution sandbox. No symptom data, search logs, or prediction outcomes are transmitted over the network or saved on external servers, ensuring 100% user confidentiality.

Engine Workflow

Nova Health runs a multi-model consensus prediction pipeline. Once at least three symptoms are registered, three local classifiers evaluate the inputs in parallel:

  • Symptom Input: Symptoms are registered manually via the search bar or by selecting anatomical regions on the interactive SVG outline.
  • Parallel Execution: The selected binary feature vector is sent to three mathematical classification models running concurrently in JavaScript.
  • Consensus Aggregation: An ensemble aggregator displays individual model confidence scores and routes the user to relevant clinical specialties and precautions.

Machine Learning Engine Architecture

Bernoulli Naive Bayes

P(C|x) ∝ P(C) ∏ P(xᵢ|C)

Evaluates joint conditional probabilities of independent symptom occurrences. Exceptionally reliable for sparse binary feature vectors.

CART Decision Tree

Gini = 1 - ∑ P(i)²

Traverses hierarchical question nodes based on Gini impurity indices, tracing exact decision paths down to leaf probability maps.

Logistic Regression

Softmax(z) = eᶻ / ∑ eᶻ

Fits linear decision hyperplanes with L2 regularization penalty, feeding scores into Softmax to yield fully calibrated likelihood bounds.
